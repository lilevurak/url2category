How to Connect to Network Applications

Back

Please see the instructions below if you need to access any of the following Taft network applications. SimBio (Virtual Labs); Mitosis; Melosis; Graphical Analysis; Fathom 2; Geometer's Sketchpad.

Windows Vista/7 Instructions:

1. Click on Start.

2. Right click on Computer.

3. Left click on Map Network Drive.

4.  Use the drop-down menu to select the drive letter you will be using.  You can choose any drive letter you like that isn't already in use.

5. In the Folder field, type in \\fs2\Applications

6. Check off "Reconnect at logon" and "Connect using different credentails".

7. Click Finish.

8. When prompted for you network password, enter your Taft network username and password in the following format.

taftschool\username *ex. taftschool\jsmith

9. Click Ok.

 

Windows 8 Instructions:

1.  On the Start screen, type the word computer.  Right-click on the Computer icon.  Then click Map network drive.

2.  Use the drop-down menu to select the drive letter you will be using.  You can choose any drive letter you like that isn't already in use.

3.  In the Folder field, type in \\fs2\Applications

4.  Check off "Reconnect at sign-in" and "Connect using different credentials".

5.  Click Finish.

6.  When prompted for your network password, enter your Taft network username and password in the following format.

taftschool\username  *ex. taftschool\jsmith

7.  Click Ok.

 

Mac Instructions:

1. With the Finder active, from the Go menu, click on Connect to Server...

2. In the Connect to Server window that opens, type in afp://xs2/Applications

*note, the share name is case sensitive so make sure that you type it in exactly as you see it above.

3. Click Connect.

4. In the authentication window that appears, enter your Taft network username and password.

username *ex. jsmith

5. Click Connect.
